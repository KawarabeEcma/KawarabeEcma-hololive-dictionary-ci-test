# ホロライブ用語のIME用追加辞書

ホロライブ、イノナカミュージック、ホロスターズの関連用語の辞書ファイルです。

## 使い方

- [Windows10](WINDOWS.md)
- [macOS](MACOS.md)

## 変換の法則

- 推しマーク - `：`の後に名前またはあだ名で変換
- タグ - `＃`の後に名前またはあだ名で変換
- ファンネーム - `〜`の後に名前またはあだ名で変換
- ツイッターID - `＠`の後に名前またはあだ名で変換

例：`：きりゅうここ`, `：ここ`, `：かいちょう`, `＃ぺこら`, `＃とわさま`, `＠ほしまち`, `〜すいちゃん`
